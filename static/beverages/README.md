# Beverage Images

Add beverage images here with the following filenames:

- `ipa.png` - IPA beer
- `coors-light.png` - Coors Light beer
- `negroni.png` - Negroni cocktail
- `bourbon.png` - Bourbon whiskey
- `smokey-scotch.png` - Smokey Scotch whiskey
- `chocolate-milk.png` - Chocolate Milk

Recommended size: 60-70px height, transparent background PNG format.
